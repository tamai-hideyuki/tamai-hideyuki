# 前提知識 — vi-assembly

- vi×アセンブリ修行の出発点

---

## 1. vi（Vim）基礎操作
- **モード切替**
    - `i`：挿入モードへ
    - `Esc`：ノーマルモードへ
    - `:`：コマンドモードへ
- **カーソル移動**
    - `h/j/k/l`：左右上下へ
    - `w`, `b`：単語単位移動
    - `0`, `$`：行頭・行末
- **編集／保存／終了**
    - `dd`, `yy`, `p`：行削除・コピー・貼り付け
    - `/文字列`, `n`：検索・次へ
    - `:w`, `:q`, `:wq`：保存・終了
- **外部コマンド実行**
    - `:!make`, `:!as -o out.o %`, `:!ld -o out %:r.o` など

---

## 2. アセンブリ言語基本構文（x86_64, AT&T記法）
- **命令フォーマット**
```asm
命令ソース, 宛先   # 例: mov $5, %rax
```
- **即値・レジスタ・メモリ参照**
  - $10：即値（イミディエイト）
  - %rax, %rdi：レジスタ
  - -16(%rbp)：スタック上のメモリ
- **ラベル定義**
  - loop_start: のように行頭で定義
---

## 3. レジスタとメモリのモデル

| 種類           | 用途例               |
| ------------ | ----------------- |
| `%rax`       | システムコール番号 / 戻り値   |
| `%rdi, %rsi` | 第1～第2引数           |
| `%rdx`       | 第3引数 / 余りレジスタ     |
| `%rsp`       | スタックポインタ          |
| `%rbp`       | フレームポインタ（呼び出し基点）  |
| メモリ領域        | スタック・ヒープ・データセクション |

## 4. セクションとディレクティブ

- **.section .data**
  - グローバル変数や文字列データを置く

- **.section .bss**
  - 初期化なしデータ（領域確保のみ）

- **.section .text**
  - 実行コード（命令）

- **ディレクティブ例**
  - .globl _start：エントリポイント公開
  - .ascii, .asciz：文字列埋め込み
  - .lcomm buffer, 16：BSSに16バイト領域

## 5. システムコール（syscall）の仕組み

1. レジスタにセット
  - %rax に syscall 番号
  - %rdi～%rsi～…に引数

2. syscall 命令 で OS に制御を渡す

3. 戻り値は %rax に返る
4. よく使う番号

| 名前   | 番号    | 用途          |
| ---- | ----- | ----------- |
| `1`  | write | 標準出力への書き込み  |
| `0`  | read  | 標準入力からの読み込み |
| `2`  | open  | ファイルオープン    |
| `3`  | close | ファイルクローズ    |
| `60` | exit  | プログラム終了     |

## 6. ビルド & 実行ツール

- **アセンブル**
```bash
as -o prog.o prog.s 
```

- **リンク**
```bash
ld -o prog prog.o
```

- **Makefile例**
```make
prog: prog.o
	ld -o prog prog.o

prog.o: prog.s
	as -o prog.o prog.s 
```

- **一発実行**
```bash
make prog && ./prog
```

## 7. デバッグ・解析

- **逆アセンブル**
```bash
objdump -d prog
```

- **バイナリ内容確認**
```bash
hexdump -C prog
```

- **GDBでステップ実行**
```bash
gdb ./prog

(gdb) break _start

(gdb) run

(gdb) stepi

(gdb) info registers
```
